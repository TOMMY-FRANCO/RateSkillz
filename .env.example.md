## Environment Variables Documented

### Client-Side (in `.env` file)

#### Required
1. **VITE_SUPABASE_URL** - Supabase project URL
2. **VITE_SUPABASE_ANON_KEY** - Public anonymous key for client operations
3. **VITE_APP_URL** - Public URL for the application

#### Optional
4. **VITE_RECAPTCHA_SITE_KEY** - Google reCAPTCHA site key for bot protection

### Server-Side (in Supabase Dashboard)

#### Auto-Provisioned
- **SUPABASE_URL** - Same as VITE_SUPABASE_URL
- **SUPABASE_ANON_KEY** - Same as VITE_SUPABASE_ANON_KEY
- **SUPABASE_SERVICE_ROLE_KEY** - Admin key (never expose in frontend!)
- **SUPABASE_DB_URL** - Direct database connection

#### Optional (must be configured manually)
1. **STRIPE_SECRET_KEY** - Stripe API secret key for payments
2. **STRIPE_WEBHOOK_SECRET** - Webhook signing secret for Stripe events
3. **RECAPTCHA_SECRET_KEY** - Server-side reCAPTCHA verification
4. **RESEND_API_KEY** - Resend API key for transactional emails

## Security Considerations

### Public (Safe to Expose)
- `VITE_SUPABASE_URL` - Public project URL
- `VITE_SUPABASE_ANON_KEY` - Protected by RLS policies
- `VITE_RECAPTCHA_SITE_KEY` - Public site key
- `VITE_APP_URL` - Public application URL

### Secret (Never Expose)
- `SUPABASE_SERVICE_ROLE_KEY` - Full database access
- `STRIPE_SECRET_KEY` - Payment processing
- `STRIPE_WEBHOOK_SECRET` - Webhook verification
- `RECAPTCHA_SECRET_KEY` - Server-side verification
- `RESEND_API_KEY` - Email sending
